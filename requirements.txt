PREREQUISITES
================================================================================
- Node.js (v18 or higher recommended)
- npm (comes with Node.js) or yarn
- MongoDB (for backend database)
- Firebase account (for authentication)

INSTALLATION - FRONTEND
================================================================================
1. Navigate to frontend directory:
   cd netflix-clone

2. Install all dependencies:
   npm install

3. Start development server:
   npm run dev

4. Build for production:
   npm run build

INSTALLATION - BACKEND
================================================================================
1. Navigate to backend directory:
   cd backend

2. Install all dependencies:
   npm install

3. Create .env file with required variables:
   PORT=5000
   MONGODB_URI=your_mongodb_connection_string
   FIREBASE_PROJECT_ID=your_firebase_project_id
   FIREBASE_PRIVATE_KEY=your_firebase_private_key
   FIREBASE_CLIENT_EMAIL=your_firebase_client_email

4. Start development server:
   npm run dev

5. Or start production server:
   npm start

RUNNING BOTH FRONTEND AND BACKEND TOGETHER
================================================================================
From the backend directory, you can run both simultaneously:
   cd backend
   npm run dev:all

This will start:
- Backend on http://localhost:5000 (or your configured PORT)
- Frontend on http://localhost:5173 (Vite default)

ADDITIONAL NOTES
================================================================================
- Ensure MongoDB is running before starting the backend
- Firebase project configuration required for authentication
- EmailJS configuration needed for contact/support features
- The project uses concurrently to run both frontend and backend together
- Default frontend port: 5173 (Vite)
- Default backend port: 5000

TROUBLESHOOTING
================================================================================
1. If npm install fails, try:
   npm cache clean --force
   npm install

2. If ports are already in use:
   - Frontend: Change in vite.config.js
   - Backend: Change PORT in .env file

3. For Firebase issues, ensure your service account key is properly configured
